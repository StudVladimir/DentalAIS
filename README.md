# DentalAIS

Automated information system for dental centers.

## Description

DentalAIS is a specialized information system designed to automate dental clinic operations. The system is built with C# using Cocoa framework and Entity Framework, optimized for legacy macOS platforms.

## About the Project

The system was developed for use by the "Zdorovye" dental clinic and served as the subject of a thesis work in the "Business Software Solutions" specialization.

## Features

The system is designed for comprehensive automation of dental clinic processes, including patient management, appointments, medical records, and administrative tasks. 

It also includes special functionality such as the ability to create and store **dental formulas** for patients in the database.
